Intent Classifier
LLM prompt → classifies query into one of three intent types
Mistral 7B
— Intent Classification —
🌐 General Info
Direct conversational queries — no external data needed
Prompt sent directly to LLM with system context
e.g. "What is a data center?" / "How does colocation work?"
📚 Knowledge (RAG)
Company-specific queries resolved via PDF knowledge base
Query embedded via nomic-embed-text
Semantic search on FAISS vector index
Top-K chunks injected into LLM context
e.g. "What's our SLA for Tier-3 racks?"
⚙️ Action
Operational requests that trigger backend API calls
LLM extracts intent + parameters from query
Maps to backend API endpoint + payload
API result returned as structured data
e.g. "Show my current power usage" / "Raise a ticket"
🤖
LLM Response Generation
Llama / Mistral 7B synthesizes a final answer from: query + retrieved context (RAG chunks or API data)
Mistral 7B
